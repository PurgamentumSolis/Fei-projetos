Como Jogar:

Recomendado para dar zoom out ate o jogo couber na tela e aparecer os dois botao

Quando o botao START for pressionado o jogo iniciara, caso perder um jogo novo iniciara imediatamente apos de mostrar sua pontuacao.
O botao CANCELAR vai parar o jogo e sera necessario um refresh na pagina para jogar o jogo

Movimento :
Teclas W e S para mexer os dois bastoes verde para cima e para baixo, A e D para mexer os bastoes vermelhos de direita a esquerda

O objetivo do jogo e sobreviver com a bola dentro da area azul o maior tempo possivel sem a bola bater nas paredes
